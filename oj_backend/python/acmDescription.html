Description

Given a string s, return the longest palindromic substring in s.

Input

The first line contains a string s. The second line contains an integer representing additional parameters.

Output

Output the required result according to the problem description.

Example 1:

Input:

s = "babad"

Output:

"bab"

Explanation:

"aba" is also a valid answer.

Example 2:

Input:

s = "cbbd"

Output:

"bb"

Note

  • 1 ≤ s.length ≤ 1000
  • s consist of only digits and English letters.
